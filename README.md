## What I Built

Hey everyone! I just finished learning how to use **Advanced Filter** in Excel, and I'm really excited to share what I learned. I worked with a sales data file that has 200 rows of information about different products sold online and in stores.

## About My Data

My Excel file has these columns:
- **Date** - When the sale happened
- **Product_ID** - Each product's number
- **Product_Category** - Like Clothing, Books, etc.
- **Sales_Channel** - Either Online or Offline (physical store)
- **Units_Sold** - How many items were sold
- **Unit_Price** - Price of one item
- **Total_Sales** - Total money from that sale

## What I Learned

### The Basics
I learned how to set up Advanced Filter, which is like the regular filter in Excel but much more powerful! Here's what I did:

- Put all my sales data in one area (200 rows!)
- Created a small "criteria box" where I write what I want to find
- Set up another area where Excel puts the filtered results

### Filtering with Multiple Conditions
The cool thing is I can filter by more than one thing at once! In my practice, I filtered to find:
- Sales that happened **Offline** (in physical stores)
- AND were in the **Clothing** category

So Excel showed me only the clothing items that were sold in stores, not online.

### Why This is Better Than Normal Filter
The regular filter button we usually use is great, but Advanced Filter can do things like:
- Copy the filtered data to a completely different spot in your sheet
- Keep your original data untouched
- Handle really complicated filter rules
- Work faster with huge amounts of data

I can see using this for:

## Real Example from My Project

By filtering for "Offline + Clothing," I could answer questions like:
- Which clothes sell better in physical stores?
- How much money did we make from in-store clothing sales?
- Do people buy more clothing items online or in stores?
